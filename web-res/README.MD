# Diseña un Blog Responsive:

1. Layout usando Grid CSS y Flexbox para componentes. Usa Grid CSS para el layout principal y flexbox para los componentes. Tendremos un diseño principal con Header+Menu, Main y Footer. Deja fijo Header (usando sticky) Usa el diseño que se aporta en la imagen (esto es para main artículos). Los artículos serán tarjetas (cards)
2. Media queries, con breakpoints para 3 dispositivos: mobile, tablet y desktop
3. Imágenes responsives
4. Tipografía con ritmo vertical y escala tipográfica (con variables CSS) adecuado entre los dispositivos (usa clamp)
5. Tabla responsive
6. Formulario responsive
7. Componentes a incluir:
- Menú responsive con botón hamburguesa
- Conjunto de etiquetas (tags)
- Paginador
- Sistema de cards
- Menú de redes sociales (con iconos, no uses texto)
8. Tened en cuenta usabilidad, accesibilidad y diseño BEM
9. Usa la metodología Mobile First
10. El contenido puede ser relleno de texto, pero puedes usar alguna temática
